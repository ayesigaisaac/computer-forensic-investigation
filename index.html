Task 1 Details

Title: Using Autopsy to retrieve deleted files

Aim: Use Autopsy to recover and analyze deleted files from a disk image.

Direct Links

  • Task 1 Folder
  • Local README

Workflow

  1. Create a new case in Autopsy.
  2. Add the disk image or local disk as a data source.
  3. Run ingest and wait for processing to complete.
  4. Open File Views -> Deleted Files.
  5. Locate target deleted files and extract them.
  6. Store extracted files in your evidence output folder.
